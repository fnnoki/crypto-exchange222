Использование FastAPI позволяет реализовать асинхронную обработку запросов, автоматическую валидацию данных, управление доступом и взаимодействие с базой данных.

Программный продукт разработан на языке программирования Python. Выбор данного языка обусловлен его высокой производительностью, развитой объектно-ориентированной моделью и большим количеством готовых библиотек, упрощающих процесс разработки и сопровождения программного обеспечения.

Для организации взаимодействия приложения с базой данных используется технология SQLAlchemy. Данная технология представляет собой ORM-средство, позволяющее выполнять работу с базой данных через объектную модель приложения без необходимости написания большого количества SQL-запросов. Для хранения данных выбрана СУБД SQLite. Данное решение отличается простотой настройки, отсутствием необходимости в отдельном серверном процессе и достаточной надёжностью. Использование SQLite позволяет обеспечить эффективное хранение информации о пользователях, заказах, курсах криптовалют, транзакциях и сообщениях чата.

Для реализации обмена сообщениями и обновления информации в режиме реального времени используется технология WebSocket. Применение данной технологии позволяет организовать мгновенную передачу сообщений между клиентами и службой поддержки без необходимости постоянного обновления страницы браузера.

Для обеспечения безопасности хранения учётных данных пользователей используется библиотека hashlib. Данная библиотека выполняет хеширование паролей перед сохранением в базу данных, что существенно повышает уровень защиты персональных данных пользователей.

Для интеграции с блокчейн-сетью Solana используется библиотека @solana/web3.js. Данная библиотека позволяет выполнять подключение к криптокошельку Phantom, создавать и подписывать транзакции, а также проверять статус подтверждения в блокчейн-сети.

Пользовательский интерфейс разработан с использованием технологий HTML5, CSS и JavaScript. Для обеспечения адаптивности интерфейса применяются медиа-запросы CSS. Разработанная автоматизированная система представляет собой веб-приложение, функционирующее на сервере и предоставляющее доступ пользователям через веб-браузер. Для установки системы необходимо выполнить следующие шаги: установить интерпретатор Python версии 3.10 или выше, установить зависимости из файла requirements.txt с помощью менеджера пакетов pip, настроить файл .env с необходимыми параметрами и запустить приложение командой python main.py.

После запуска система становится доступной по адресу http://localhost:8000. Для начала работы пользователю достаточно перейти по данному адресу с использованием любого современного браузера. Дополнительная установка программного обеспечения на рабочие места пользователей не требуется. Все вычисления и обработка данных выполняются на серверной стороне приложения, а результаты отображаются в интерфейсе веб-сайта.

После запуска системы выполняется проверка доступности базы данных и основных сервисов приложения. При успешном запуске открывается главная страница сайта, содержащая форму создания заказа и информацию о текущих курсах криптовалют. Основной целью тестирования являлась проверка корректности работы программного обеспечения, соответствия реализованного функционала требованиям технического задания и выявление возможных ошибок в процессе эксплуатации.

В ходе проверки выполнялось тестирование пользовательских сценариев, связанных с созданием заказов, получением курсов, оплатой через криптокошелёк, проверкой транзакций, обменом сообщениями в чате и выполнением операций с базой данных. Особое внимание уделялось корректности сохранения информации, отображению данных в пользовательском интерфейсе и разграничению прав доступа между различными категориями пользователей.

Дополнительно была выполнена проверка работы системы в различных браузерах. Результаты тестирования подтвердили корректное отображение страниц и стабильную работу основных функций веб-приложения. Ошибок, препятствующих эксплуатации программного продукта, обнаружено не было.

По результатам проведённого тестирования установлено, что разработанная автоматизированная система обеспечивает выполнение предусмотренных функций и может использоваться для автоматизации деятельности сервиса обмена криптовалют.

**2.10 Руководство оператора**

**2.10.1 Назначение программы**

Автоматизированная информационная система обмена криптовалют предназначена для автоматизации деятельности сервиса по обмену цифровых активов и организации взаимодействия между администрацией и клиентами.

Программа обеспечивает хранение и обработку информации о заказах, курсах криптовалют, транзакциях и обращениях в службу поддержки. Система позволяет администраторам управлять заказами, отслеживать статусы оплат и контролировать деятельность сервиса.

Для клиентов предусмотрены возможности просмотра курсов криптовалют, создания заказов на обмен, оплаты через криптокошелёк Phantom и общения со службой поддержки через встроенный чат.

Использование программы позволяет сократить затраты времени на выполнение рутинных операций, повысить эффективность управления сервисом и улучшить качество обслуживания клиентов.

**2.10.2 Условия выполнения программы**

Программа функционирует в виде веб-приложения и не требует установки дополнительного программного обеспечения на рабочее место пользователя. Для работы с системой необходим персональный компьютер, ноутбук, планшет или другое устройство, имеющее доступ к сети Интернет.

Для корректной работы программы рекомендуется использовать современные веб-браузеры, такие как Google Chrome, Microsoft Edge, Mozilla Firefox или Opera. Работа системы возможна под управлением операционных систем семейства Windows, macOS и Linux.

Для комфортной работы рекомендуется наличие не менее 4 ГБ оперативной памяти и стабильного подключения к сети Интернет. Для оплаты через криптокошелёк Phantom необходимо установить соответствующее расширение для браузера.

Серверная часть приложения функционирует на базе FastAPI и использует систему управления базами данных SQLite для хранения информации.

**2.10.3 Выполнение программы**

Для начала работы необходимо открыть веб-браузер и перейти по адресу размещения автоматизированной системы. После загрузки главной страницы пользователю становятся доступны форма создания заказа и текущие курсы криптовалют.

Для создания заказа необходимо выбрать актив (USDT или SOL), ввести сумму, выбрать валюту и банк для получения средств. Система автоматически рассчитает сумму к получению с учётом текущего курса и комиссии сервиса. После нажатия кнопки «Отправить» будет создан заказ и сгенерирован адрес для оплаты.

После создания заказа пользователю отображаются платёжные реквизиты. Для оплаты необходимо отправить криптовалюту на указанный адрес. При наличии установленного расширения Phantom можно воспользоваться кнопкой «Оплатить SOL через Phantom» для отправки средств напрямую через интерфейс системы.

Для обращения в службу поддержки необходимо нажать на кнопку чата в правом нижнем углу экрана. После нажатия на кнопку «Начать чат» открывается окно чата, где можно вводить и отправлять сообщения. Ответы от администрации отображаются в режиме реального времени.

Администратор системы после авторизации получает доступ к панели управления. Ему доступны операции по просмотру списка заказов, фильтрации по статусам, просмотру детальной информации и управлению чатами с клиентами.

**2.10.4 Сообщения оператору**

При возникновении ошибок система отображает соответствующие уведомления. При создании заказа с незаполненными обязательными полями выводится сообщение об ошибке. При неверном токене администратора доступ к панели управления запрещается. При отсутствии подключения к блокчейн-сети отображается соответствующее предупреждение.

**2.11 Политика информационной безопасности**

**2.11.1 Ограничение прав доступа пользователей базы данных**

Одним из основных требований к разработанной автоматизированной информационной системе является обеспечение защиты хранимой информации и предотвращение несанкционированного доступа к данным. Для решения данной задачи в системе реализован механизм разграничения прав доступа пользователей.

Доступ к функциональным возможностям приложения определяется ролью пользователя. После успешной авторизации система автоматически предоставляет пользователю доступ только к тем разделам и операциям, которые предусмотрены его правами.

Клиенты имеют возможность просматривать курсы криптовалют, создавать заказы, оплачивать их через криптокошелёк и общаться со службой поддержки. Доступ к функциям изменения системных данных для данной категории пользователей ограничен.

Администратор обладает расширенными правами доступа и может выполнять управление заказами, просмотр транзакций, управление чатами и другими объектами системы. Доступ к административным разделам осуществляется только после успешной проверки полномочий пользователя с использованием секретного токена.

Разграничение прав доступа позволяет исключить возможность случайного или преднамеренного изменения данных пользователями, не обладающими соответствующими полномочиями, а также обеспечивает сохранность информации, содержащейся в базе данных.

**2.11.2 Авторизация и регистрация пользователей**

Для получения доступа к функциональным возможностям системы администратор должен пройти процедуру авторизации.

При авторизации администратор вводит секретный пароль, после чего система выполняет его проверку. В случае успешной аутентификации генерируется токен доступа, который сохраняется в браузере и используется для последующих запросов к защищённым разделам приложения.

Для повышения уровня безопасности пароль администратора не хранится в открытом виде. Проверка выполняется с использованием механизма HMAC-SHA256, что обеспечивает защиту от перехвата учётных данных. Благодаря этому даже при получении доступа к исходному коду приложения невозможно определить исходное значение пароля.

При вводе некорректных данных система уведомляет пользователя о невозможности выполнения входа и запрещает доступ к защищённым разделам приложения. Использование механизмов авторизации и токенов доступа позволяет обеспечить необходимый уровень защиты данных и безопасную эксплуатацию автоматизированной информационной системы.
